# 1024-simple-things
The web app is just 1024 simple things to do

## Administrator Dashboard
- authentication
  - login page
  - registration with CLI or by Super Administrator
- authorization (ACL)
- password recovery via e-mail
- 2FA

## Customer account
- authentication
  - login page
  - registration page
- welcome email
- password recovery
- 2FA
- Social Login

## Mailing
- integration with 3rd party mailing provider
